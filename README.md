# Счетчик на Vite + React + TypeScript

Простое приложение-счетчик с горячей перезагрузкой из коробки.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение откроется на http://localhost:5173

**Горячая перезагрузка работает автоматически!** Просто сохраните файл и изменения применятся мгновенно без перезагрузки страницы.

## Сборка

```bash
npm run build
```

## Деплой на GitHub Pages

Проект автоматически деплоится на GitHub Pages при пуше в ветку `main`.

После пуша:
1. Перейдите в Settings → Pages в вашем репозитории
2. Выберите Source: "GitHub Actions"
3. После успешного выполнения workflow приложение будет доступно по адресу: `https://dikanevn.github.io/TC/`

Или просто запушите код в `main` ветку - деплой произойдет автоматически!
